Privacy Policy

MNHN Specimen Extractor is a Chrome Extension developed to assist researchers in analyzing scientific articles and pushing extracted specimen data to Zotero.

Data Stored

The extension allows the user to locally store:

  • OpenAI API Key (used for AI-powered text analysis)
  • Zotero API Key and Library ID (used to push data to your Zotero library)

How Your Data Is Used

These keys are stored only in your browser using chrome.storage.local. They are not sent to any third party, and are used exclusively by the extension to fulfill its core function: interacting with the OpenAI API and the Zotero API on your behalf.

User Control

You may delete your API keys at any time using the extension's settings interface. No personal data is collected, sold, or shared.
